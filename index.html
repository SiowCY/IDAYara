IDA Pro with Yara Scan

This is Yara Scan for IDA Pro without PySide installed

First download IDAScope from IDAScope v11. Please ensure you have install yara-python, you may find it on Yara-Python

For Windows user please copy C:\python27\Lib\site-packages\yara.pyd into your \IDA\lib folder.

For Linux please refer to http://secshoggoth.blogspot.com/2014/02/installing-yara-into-ida-pro-64-bit.html

In order to use Yara Scan without PySide, please add a new script file and locate it on same directory with IDAScope: IDAYara.py

If you want to further look for the detected location of the address, you can download the edited version of YaraScanner.py

The changes is made on _result_callback function.

Please note the scans may take few minutes depends on the yara rules.
